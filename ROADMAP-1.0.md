# GeoSub 1.0 本地发布验收清单

更新日期：2026-07-10

## 发布原则

- 1.0 之前先在本地完成开发和验收，不再把每个小功能都推送到 GitHub 或服务器。
- 线上候选版本更新为 `0.3.0-beta.14`；本次用于同步本地、GitHub 与服务器，正式 `1.0.0` 仍需完成剩余验收项。
- 所有 1.0 候选改动必须先通过本地检查，再进入最终发布流程。
- 1.0 发布时再统一定版本、写更新日志、推送 GitHub、升级服务器。

## 当前阶段

目标：把 GeoSub 从 beta 功能堆叠，收敛成可以正式上线的 `1.0.0`。

当前判断：

- 前台核心页面已经成型，但仍需要完整移动端、中英文和内容入口验收。
- 后台采集、审核、诊断已经有闭环雏形，但还需要降低理解成本和确认所有状态可靠。
- App Store 价格源、汇率、税务、风险和购买力模型已经接入，但仍要做数据准确性总验收。
- SEO、文章系统、站点地图、统计代码入口已经有基础，需要按正式站标准复查。
- 发布脚本和服务器部署链路可用，但 1.0 之前先停止频繁部署。
